|---|---|
| • Busca rápida por aluno | • Dependência de internet |
| • Histórico de treinos | • Falta de gráficos intuitivos |
| • Informações centralizadas | • Poucos recursos avançados no início |
| • Interface simples | • Resposta do personal pode atrasar |
| • Notificações automáticas | • Adaptação inicial dos usuários |
| • Acessibilidade incluída | • Acessibilidade incompleta no começo |

| **Oportunidades** 🚀 | **Ameaças** 🛑 |
|---|---|

| • Uso de vídeos explicativos | • Dificuldade tecnológica de usuários |

## 📌 Resumo SWOT

- **Forças:** Recursos internos positivos do sistema.  
- **Fraquezas:** Limitações que precisam melhorar.  
- **Oportunidades:** Tendências e chances de crescimento.  
- **Ameaças:** Fatores externos que podem prejudicar.
